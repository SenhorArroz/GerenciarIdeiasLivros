Para rodar o sistema, primeiro dê o comando 'composer install' para instalar as dependências do projeto.
Depois, faça o comando 'cp .env.example .env' e configure como quiser o banco de dados. 
Assim que conseguir, dê o 'php artisan migrate' para criar as tabelas no banco de dados.
Para rodar o sistema, dê o comando 'php artisan serve' e acesse o sistema com o link disponível.
